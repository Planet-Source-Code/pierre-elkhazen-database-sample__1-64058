Tutorial on how to Make a DataBase Software the Easiest and Simplest way Using ADO

 

Solution for all DataBase Beginners and Intermediates. Complete Access DataBase Sample with Add, Find, Sort, Filter, Delete, Save, Records  Navigation, SQL Queries, Reports, and Utilities. Minimum and Few Code, Simple, Easy, and Commented.

 

 

Step 1 : What you need to Start

  • 1- ADO Data Control.

  • 2- DataGrid Control (or Micosoft Hierarchical FlexGrid Control )

  • 3- Project Reference to Microsoft  ActiveX Data Object and Microsoft DAO

  • 4 - Project Reference to Microsoft DAO 3.6 Object Library

  • 5- And of course a DataBase which you can easily create in Access or create by Code.

Step 2 : Set TextBoxes & DataGrid1 Properties

  • 1- Set all TextBoxes, DTPicker1,  DataGrid1,  DataSource Properties to Adodc1.

  • 2- Set all TextBoxes, DTPicker1,  Report Designer, DataField Properties to the Appropriate Fields.

Step 3 : Make the DataBase Connection and Activate the Recordset:

 

CmdOK_Click will show you how to Connect to the DataBase. This is the only Code you need, to Activate all at once Adodc1 (Patients DB), DataGrid1 (Grid View), and all TextBoxes. You may set Adodc1 Connection Manually by Right Click Adodc1 then Click Properties then you don't need to write any Connection Code.

 

That's all what you need to Do! You now have a Sample DataBase Program.

Notes:

  • Now look under each  Command  in Patients Tab for Code: Add, Delete, Find, Sort. Also look in the Menu Items.

  • The Report Tab and Find Command will show you how to make Queries and use SQL. Also the Report Tab will show you how to make DataBase Connection and Recordsets in Code, using ADO, without the ADODC ActiveX Control.

  • To Sort, Click a Field and Click Sort.

  • DataBase Password is: 555 . If you change it make sure you remember it. If you want to Add multiple Users and Passwords, an easy way is to Add a Table with these information and check against it on Log On. And a User Administrator with Password can only change these information.

These are All the Tools you need to make a Complete and Effective DataBase Program in a simple and easy way. For an Advanced Program you may want to Enhance the Report Designer. Also you may substitute the DataGrid Control with the Micosoft Hierarchical FlexGrid Control. Adding  various SQL Statements  and Filters you can get Advanced  DataBase Manipulations Results. Later on you may experience with the Data Environment if you wish.

Feel Free to use this Sample Program in any way you want except for the Logo.
